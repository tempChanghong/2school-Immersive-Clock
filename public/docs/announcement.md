# 🎉 欢迎使用天津二中沉浸式时钟

本时钟是一款专为本校自习模式设计的轻量化全屏数字时钟工具。

## ✨ 核心特性

- **专属自习模式**：专为学习场景优化的时间管理。
- **环境监测**：实时展示天气预警，并在课时期间进行噪音监测。
- **Classworks 联动**：直接在右侧唤出当日班级作业板信息。
- **免安装体验**：支持 PWA，可在浏览器端像原生应用一样安装，体验更佳。

## 💡 小贴士

- 鼠标点击页面空白处，或按下键盘 `Space` / `Enter` 键即可唤出控制台。
- 可以点击右下方的设置按钮，根据个人偏好调整。
