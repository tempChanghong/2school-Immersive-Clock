# 常见问题 (FAQ)

## 📱 安装与启动

### Q: 如何将时钟安装到桌面或手机？

**A:** 本应用采用 PWA (渐进式 Web 应用) 技术，支持跨平台安装：

- **Windows / macOS (Chrome/Edge)**：点击浏览器地址栏右侧的“安装”图标（通常为电脑/加号图标），确认即可。
- **Android (Chrome)**：点击浏览器菜单 -> “安装应用”或“添加到主屏幕”。
- **iOS (Safari)**：点击底部“分享”按钮 -> 上滑找到并点击“添加到主屏幕”。
- **桌面客户端**：我们也提供打包好的 Electron 安装包，可前往 GitHub Releases 下载。

### Q: 为什么我看不到安装按钮？

**A:**

1. 请确认您使用的是现代浏览器（Chrome 80+, Edge, Safari）。
2. PWA 仅在 HTTPS 环境下生效（本地 `localhost` 除外）。
3. 如果您之前已安装过，安装按钮可能会隐藏，请在应用列表中查找。

## ✨ 功能与设置

### Q: 如何开启多个倒计时轮播？

**A:**

1. 进入**设置面板** -> **倒计时管理**。
2. 确保已添加至少两个倒计时事件。
3. 在“倒计时设置”区域，勾选 **“开启多事件自动轮播”**。
4. 您还可以调整下方的 **“轮播间隔”**（默认为 10 秒）。

### Q: 如何自定义背景图片？图片会上传吗？

**A:**

1. 进入**设置面板** -> **自习设置** -> **背景设置**。
2. 选择 **“自定义图片”**，点击上传按钮选择本地图片。
3. **隐私说明**：您上传的图片**仅保存在您的浏览器本地**（IndexedDB），绝对不会上传到任何服务器，请放心使用。

### Q: 自习模式界面太乱，可以隐藏部分组件吗？

**A:** 当然可以。在**设置面板** -> **自习设置** -> **显示设置**中，您可以自由开关以下组件：

- 状态栏
- 噪音监测器
- 倒计时组件
- 励志语录
- 大字时间
- 日期显示

### Q: 励志语录的来源可以修改吗？

**A:** 可以。在**设置面板** -> **内容设置**中，您可以管理语录频道。支持开启/关闭特定频道（如文学、哲学、动画等），并可以调整每个频道的出现权重。

## 🔧 故障排查

### Q: 无法获取天气或一直显示“定位中”？

**A:**

1. **权限检查**：请确保浏览器已授予本站“位置信息”访问权限。
2. **手动刷新**：在天气卡片上点击“刷新”图标，系统会尝试重新定位。
3. **网络问题**：如果 IP 定位服务无法访问，可能导致定位失败，此时将显示默认城市。

### Q: 噪音监测没有数据或波形不动？

**A:**

1. **麦克风权限**：首次开启时浏览器会请求麦克风权限，请务必点击“允许”。
2. **设备检查**：请确保您的电脑/手机麦克风硬件正常且未被静音。
3. **系统设置**：在操作系统（Windows/macOS）的隐私设置中，确保浏览器被允许访问麦克风。

### Q: 倒计时结束没有提示音？

**A:**

1. **自动播放限制**：现代浏览器默认阻止自动播放音频。请尝试先与页面进行交互（如点击任意按钮）。
2. **静音模式**：检查系统音量，并确认未开启“请勿打扰”模式。
3. **文件检查**：提示音文件为 `public/ding.mp3`，请确保网络环境能正常加载该资源。

### Q: HUD (控制栏) 不显示或很快消失？

**A:** HUD 设计为沉浸式自动隐藏（约 8 秒无操作）。

- **唤出**：点击页面空白处，或按键盘 `Space` / `Enter` 键。
- **注意**：当打开模态框（如设置、倒计时编辑）时，HUD 将暂时禁用。

## 🛡️ 数据与隐私

### Q: 更换浏览器或设备后，我的设置还在吗？

**A:** **不在**。
本应用是一款纯前端应用，所有数据（设置、倒计时、自定义背景等）均存储在您当前浏览器的 **LocalStorage** 和 **IndexedDB** 中。更换浏览器、设备或清除浏览器缓存均会导致数据丢失。

### Q: 如何重置所有设置？

**A:** 您可以清除浏览器的站点数据，或者在浏览器的开发者工具中手动清除 LocalStorage。未来版本将提供一键重置功能。
